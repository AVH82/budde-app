# Assets graphiques

Organisation des fichiers visuels de Budd€.

## Structure

- `buddy/` — mascotte Buddy.
- `icons/` — icônes générales.
- `ui/` — éléments d'interface.
- `objects/` — objets illustrés.
- `stickers/` — stickers et visuels décoratifs.
- `merchants/` — logos et visuels commerçants.
- `backgrounds/` — fonds et textures.
- `fonts/` — polices locales éventuelles.
- `sounds/` — sons d'interface.
- `logo/` — logo Budd€ existant.
- `nav/` — icônes de navigation existantes.

## Conventions

- Noms en kebab-case.
- Ne pas modifier un PNG existant lors d'un simple déplacement.
- Ajouter un README dans les sous-dossiers complexes.
